# Estrutura de jogos

O Code Hack ainda vive nos arquivos atuais para evitar regressões enquanto a plataforma Meira Games nasce.

Novos jogos devem ganhar pastas próprias em `src/games/<id-do-jogo>` e módulos equivalentes no backend, reaproveitando somente a camada comum de salas, perfil, autenticação, mensagens e imagens.
