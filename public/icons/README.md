# Icones do jogo

Substitua estes arquivos PNG mantendo os mesmos nomes:

- `logo.png`: logotipo usado na tela inicial e no cabecalho.
- `vida.png`: icone de vidas no placar.
- `descriptografia.png`: icone de descriptografias/interceptacoes no placar.
- `confirmado.png`: icone de confirmacao. O estado "nao confirmado" usa este mesmo arquivo com opacidade baixa via CSS.
- `lider.png`: icone do lider da sala.

O favicon fica em `../favicon.png`.
